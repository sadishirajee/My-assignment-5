Important FAQ

What is the Document object model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each part of the document (like elements, attributes, and text) is an object that can be manipulated using programming languages like JavaScript.

How do you select an element from DOM?

You can select an element from the DOM using methods like document.getElementById("id"), document.querySelector("selector"), or document.getElementsByClassName("class").

What is the relevent delegation in the context of the DOM, and why it is useful?

delegation in the DOM is a technique where a single event listener is attached to a parent element to manage events from its child elements. It's useful for handling events efficiently

How do you manipulate an elements attributes and styles using the DOM?

using element.setAttribute("attribute", "value") one can manipulate an element's attributes.
